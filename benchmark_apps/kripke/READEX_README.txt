# README READEX for Kripke

 - for compilation use one of compile_for_XXX.sh scripts, compiled code is located in the build directory
 - to run the code with the tools there are bash scripts in scripts directory
   compilation scripts make a copy of appropriet script/s in the build directory, according the compiled version of the code
 - the already computed results are located in the RESULTS directory 
   if you want to skip any of the steps you may make a copy of a file you need a name it correctly
  

 - for a compilation of new version, empty build directory is allways made
   store your results out of this directory !!!
